alpha버전 테스트는 Windows-64bit 운영체제에서 진행되었고 다른 운영체제는 테스트해보지 않았다(리눅스에서는 VideoCapture(0)를 VideoCapture(1)로 해야 작동된다고 SYH가 말했던 것 같긴 하다).

numpy, opencv, mediapipe, pynput, pyautogui가 설치되어있어야 한다.

main.py : 실행시키면 CAT이 작동된다.
mcat.py : main.py에서 사용되는 모듈이다. main.py와 같은 폴더 내에 있어야만 정상작동한다.

[메뉴얼-손 모양에 따른 동작]
- 기본적으로 오른손을 사용한다
아래는 접힌 손가락에 따른 동작

● 약지(반지손가락?),소지(새끼손가락) : 커서 이동(손을 이동)
● 엄지,약지,소지 : 왼쪽 마우스 누름+커서 이동
● 엄지,검지 : 더블클릭(좌)
● 엄지,약지 : 우클릭
● 중지,약지 : 스크롤(손을 위아래로 이동)
